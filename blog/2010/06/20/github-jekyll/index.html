孤独な戦死

Jun 20th, 2010

恐ろしい。

何が恐ろしいって自分の無知が恐ろしい。

JavaScriptはライブラリ使うのやめたらサクサク書けるようになりましたとさ。 というか面白い言語だと思った。

言語を批評できるレベルではないと思うんだけど、書きやすいしオブジェクトの定義とかが細かくて面白い。 もしかしたらほかの言語で意識してなかっただけかもしれないけど。

すげえ基礎的なことだろうけど一応メモということで。 イベンドハンドラ = 関数名();　//これはアウト イベントハンドラ = 関数オブジェクト; //これはおｋ イベントハンドラ = function(){関数名();} //これもおｋ

うーむ。

イベントオブジェクトはIE以外のブラウザでは引数として渡せるんだけどIEでは無理みたいっすねー。 まぁ各々のブラウザで定義が違うのは仕方がないかな。統一してもらいたいけど。

JavaScriptつながりですが、現在Ajaxチャット開発中です。 複数人でやってるので中々作業が進みませんが、面白いですね。

WEBONAは参加者引き続き募集中です。 WEBデザインを近々したいな。 やらせてくれる人いたらコンタクトほしいっす。
