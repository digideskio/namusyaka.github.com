    TwitterのつぶやきをJavaScriptで取得して適当な位置に表示する。

    ここ数日、どこだかドアやグレイモンなど、しょうもない開発しかやっていないnamusyakaです。 久しぶりに誰かの役に立ちそうなJavaScriptでも書いてみようかなと思ったので書きます。 実際役に立つかどうかはわかんないです。 使用例

    機能

    • http://から始まる文字列を自動でリンクに変換。
    • @から始まる文字列を自動でリンクに変換。
    • #から始まる文字列を自動でリンクに変換。
    • 時間は世界標準時を日本の標準時に変換。
    • 途中で空白や改行、もしくは文の末尾に達した場合はそこでリンク終了。

    以下、JavaScriptソース

    これをツイートを表示したい位置に貼り付けると、定義リストでツイートの内容と時間が表示されます。

    114531326の部分

    は、僕のTwitterのIDなわけですが、これを調べるツールを3秒くらいで書いてみました。 絶対これよりパフォーマンスとかもよくてすばらしいツールはあると思うんですが、ググって1ページ目に出なかったので書きました。 とりあえずこれで出せると思います。 ちなみにTwitterの自分のページに飛んでソース読めば普通にわかります。

    count=5の部分

    は、取得するツイート数です。20くらいまで取得できた気がします。 5の部分変更してみるとわかると思います。

    時間表示の部分が気に入らない場合

    changeDate関数の、年、月、日、時、分、秒のところを適当に変更してみるといいと思います。 ちなみにTwitterから取得してる時間は世界の標準時なんですが、 上に書いてある通り、パンパンここJAPANなので日本時間に変更してあります。

    定義リストやめてもらいたい場合

    dl・dt・ddの部分を好きな要素に変更しちゃってください。 CSSは一切書いてないので好きにレイアウトできると思います。

    なんかもう全体的に気に入らない場合

    何が気に入らないか教えてくれるとうれしいです。
