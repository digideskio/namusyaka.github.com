僕がプログラミングを始めたきっかけ

May 17th, 2011

流行ってるのでやります。

きっかけ

きっかけは中３のときにC言語でwinsockを使った簡単なパスワードクラックプログラムを書いたことです. 所謂厨二病ってやつだった気がします. 見てると結構, 厨二病が原因だったと振り返っている人がいるので, 満更馬鹿にできたもんでもないなーと思いました.

言語

HTML
好きです.遠い昔はタグ辞典見ながらデザイン作って俺ツエーやってましたが, 今は論理マークアップをしています.
XHTML
XML自体書かないのでどうかなって思いますが, i-mode2規格にてCSSを反映させるにはXHTMLである必要があるみたいなのでその時は使います.
CSS
書きます.
JavaScript
一番使ってる言語かもしれないです.同じ人は多いみたいですね.
Ruby
サーバサイドを利用した個人のウェブアプリケーション開発にはRubyを使います.
実装の仕方が何通りもあって迷いますが, その柔軟性もメリットの一つなんだと思います.
Python
最近初めて触ってみましたが, 全然書けません.
Java
リファレンスとか見て, あー書きたいなあと思って一度テストをしたっきり使ってません.
機会があれば書こうと思います.
C
きっかけとなった言語ではありますが, ほとんど書きません.
文字列操作など, いちいちだるいのが原因だと思います.
MySQL
DBが必要なときにはMySQLを使います.あんまり詳しくないです.
