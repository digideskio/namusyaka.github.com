公開したOperatorというモジュール

Rubygems https://rubygems.org/gems/Operator Github https://github.com/namusyaka/Operator

PHP で凄く簡単に GET/POST 送信ができる関数を作りましたにあった関数をみてて思いついたものです。 get/postを簡単に出来るのはもちろん、responseを操作したい場面が多々あったのでそっちのメソッドも少し実装してます。 構造的には -Operator --Operator::HTML --Operator::HTTP ですが、HTTPはHTMLをスーパークラスにもつので基本的メソッドはHTMLで定義されています。 使い分けは、コンストラクタに既に完成されたHTMLソースを渡す場合HTML、URLを渡す場合はHTTPにっていう想定。 gemにあがっているので、以下を実行するとインストールできます。
