新生活

date: 2011-03-25 comments: true categories: その他

今月の二十日から一人暮らしをしてるんだけど、ネットがつながりません。 今までネット中心の生活だったから本当に困りました。 良い機会だから読んでない本を消化しようと試みるも一日で読み終わってしまいました。 だからほかの娯楽、もとい夢中になれることが必要で、色々考えてるけど見つからないです。

∴俺ってつまらない人間だなあと思ったという話。

プログラミング

いつも公式リファレンスなんかを読みながらプログラミングをするんですが、携帯でググりながらやるのは非常に面倒です。 せっかくサーバーを借りたのに申し訳ないですね。

来月の大学入学式が終わった時点でネットがつながってなかったらネットができる環境に移住したいと思います。
