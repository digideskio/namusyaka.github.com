    RubyでRSSを読んだりキャッシュしたりっていうクラスをつくった

    classをほとんど使わない人間なのでどう設計したもんかと大分悩みましたが上手い答えは出ませんでした。 これ以上悩んでも出そうにないのでとりあえず公開。

    まー小さいクラスです。

    updatedはcheck,あるいはcacheが返す配列(要素は全てHashオブジェクト)の中で新たに更新があったindexを配列で返します。 これに合わせてメール送信をすれば、それっぽいものが作れると思います。

    追記

    ちょいちょいアホなことをしてたので修正したのと、proxyメソッドを追加した。 後cacheメソッドはキャッシュ時間を引数に渡すようにした。デフォルトは３０分。
