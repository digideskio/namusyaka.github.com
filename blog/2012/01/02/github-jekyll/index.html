    Skypeチャットでコンタクトリストに追加されてないメンバーに承認リクエストを飛ばすscriptを書いた

    意識が高い学生のチャットに一人放り込まれた意識不明の学生namusyakaです。 大成しそうな人ばっかりですし、コンタクトを共有したいのですが全員にリクエストを飛ばすのは正直だるいです。 なのでコードを書きました。Skype4Pyが必要です。

    requestを送りたい人達がいるチャットで#sendContactRequestToChatMembersと発言すれば自動で飛ばされます。 効率的に女の子と仲良くなりたい人も使えると思います。
